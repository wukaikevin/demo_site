# 专业后台管理系统使用指南

## 🎨 设计理念

新的后台管理系统参考了业界最佳实践，采用现代化的设计语言：

- **Ant Design Pro** - 企业级中后台产品
- **Vue Element Admin** - Vue.js管理后台
- **Bootstrap Admin** - 经典管理后台
- **Django Admin** - Python Web框架管理界面

## 📊 功能模块

### 1. 工作台（Dashboard）

**统计卡片**：
- 📊 总案例数
- ⏳ 待审核数量（橙色）
- ✓ 已通过数量（绿色）
- ✗ 已拒绝数量（红色）

**实时更新**：
- 数据实时统计
- 自动刷新支持
- 可视化展示

### 2. 案例列表

**数据表格**：
- 列表式展示，适合大量数据
- 固定表头，滚动内容
- 悬停高亮效果

**表格列**：
- 选择框（复选框）
- 预览图（60x60px）
- 案例标题
- 应用ID
- 状态（标签显示）
- 提交时间
- 操作按钮

**操作按钮**：
- 查看 - 在新标签页打开详情
- 通过 - 仅待审核案例显示
- 拒绝 - 仅待审核案例显示
- 删除 - 所有案例都可删除

### 3. 搜索和筛选

**搜索栏**：
- 关键词搜索（标题、应用ID）
- 状态筛选（全部/待审核/已通过/已拒绝）
- 应用分类筛选
- 重置按钮

**搜索特点**：
- 实时搜索（可选）
- 多条件组合
- 搜索结果高亮

### 4. 批量操作

**批量操作栏**：
- 选择数量提示
- 批量通过
- 批量拒绝
- 取消选择

**全选功能**：
- 表头复选框全选当前页
- 智能状态同步
- 跨页选择保留

### 5. 分页组件

**分页信息**：
- 显示总记录数
- 当前页范围（1-20 / 共100条）
- 上一页/下一页按钮

**分页样式**：
- 页码按钮
- 激活状态高亮
- 禁用状态处理

### 6. 侧边栏导航

**菜单分组**：
- 审核管理（工作台、待审核、已通过、已拒绝）
- 数据管理（所有案例、分类管理）
- 系统设置（系统设置）

**菜单项**：
- 图标 + 文字
- 悬停效果
- 激活状态
- 点击跳转

### 7. 顶部导航栏

**左侧**：
- 面包屑导航
- 当前位置显示

**右侧**：
- 修改密码按钮
- 用户头像
- 用户名显示

### 8. 弹窗组件

**拒绝原因弹窗**：
- 文本域输入
- 必填验证
- 确认/取消按钮

**修改密码弹窗**：
- 原密码输入
- 新密码输入
- 确认密码输入
- 密码强度验证

## 🎯 使用流程

### 审核工作流

```
1. 登录后台
   ↓
2. 查看工作台统计
   ↓
3. 进入"待审核"列表
   ↓
4. 浏览案例缩略图
   ↓
5. 点击"查看"按钮预览详情
   ↓
6. 决定通过或拒绝
   ↓
7. 批量操作（可选）
   ↓
8. 完成，统计数据更新
```

### 数据管理流

```
1. 进入"所有案例"
   ↓
2. 使用搜索/筛选功能
   ↓
3. 勾选目标案例
   ↓
4. 执行批量操作
   ↓
5. 查看操作结果
```

## 🔧 技术栈

### 前端技术

- **HTML5** - 语义化标签
- **CSS3** - Flexbox、Grid、动画
- **JavaScript ES6+** - 原生JS，无框架依赖
- **Fetch API** - 网络请求

### 设计系统

**排版**：
- 字体：-apple-system, BlinkMacSystemFont, Segoe UI
- 字号：12px / 14px / 16px / 20px / 30px
- 行高：1.5

**间距**：
- 基础单位：8px
- 常用间距：8px / 16px / 24px / 32px

**圆角**：
- 按钮/输入框：2px
- 卡片：4px

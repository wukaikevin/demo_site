## 导航菜单

### 侧边栏菜单

左侧深色侧边栏包含所有功能入口：

#### 1️⃣ 工作台
```
点击"工作台"菜单
  ↓
显示全部案例统计
  ↓

```

#### 2️⃣ 待审核
```
点击"待审核"菜单
  ↓
自动筛选状态=待审核
  ↓
只显示待审核的案例
  ↓
便于批量审核处理
```

#### 3️⃣ 已通过
```
点击"已通过"菜单
  ↓
自动筛选状态=已通过
  ↓
显示已审核通过的案例
  ↓
查看已发布内容
```

#### 4️⃣ 已拒绝
```
点击"已拒绝"菜单
  ↓
自动筛选状态=已拒绝
  ↓
显示已拒绝的案例
  ↓
支持复审（重新通过）
```

#### 5️⃣ 所有案例
```
点击"所有案例"菜单
  ↓
显示全部案例（不限状态）
  ↓
支持搜索和筛选
  ↓
查看和管理所有数据
```

#### 6️⃣ 分类管理
```
点击"分类管理"菜单
  ↓
按应用ID分组统计
  ↓
显示每个应用的案例数
  ↓
点击"查看案例"可筛选该应用
```

#### 7️⃣ 系统设置
```
点击"系统设置"菜单
  ↓
显示系统信息
  ↓
查看运行状态
  ↓
快速操作入口
```

#### 8️⃣ 访问前台
```
点击"访问前台"菜单
  ↓
在新标签页打开前台首页
  ↓
查看用户视角的展示效果
```

#### 9️⃣ 退出登录
```
点击"退出登录"菜单
  ↓
清除登录状态
  ↓

```

## 页面说明

### 📊 工作台

**显示内容**：
- 四个统计卡片（总数、待审核、已通过、已拒绝）
- 完整的案例列表（所有状态）

**使用场景**：
- 快速查看整体情况
- 了解各状态案例数量
- 综合管理所有案例

### ⏳ 待审核

**显示内容**：
- 统计卡片
- 仅显示待审核状态的案例

**使用场景**：
- 每日审核工作
- 批量审核处理
- 查看待审核队列

**操作流程**：
1. 查看"待审核"数量
2. 点击"查看"按钮预览案例详情
3. 根据内容质量决定：
   - 合格 → 点击"通过"
   - 不合格 → 点击"拒绝"
4. 或使用批量操作提高效率

### ✓ 已通过

**显示内容**：
- 统计卡片
- 仅显示已通过的案例

**使用场景**：
- 查看已发布内容
- 内容质量复查
- 数据统计分析

### ✗ 已拒绝

**显示内容**：
- 统计卡片
- 仅显示已拒绝的案例

**使用场景**：
- 查看被拒绝内容
- 复审拒绝案例（点击"通过"可恢复）
- 分析拒绝原因

### 📁 所有案例

**显示内容**：
- 完整的案例列表
- 支持搜索和筛选

**使用场景**：
- 查找特定案例
- 搜索关键词
- 按条件筛选
- 批量管理

### 🏷️ 分类管理

**显示内容**：
- 按应用ID分组的统计表格
- 每个应用的数量分布
- 快速筛选入口

**表格列**：
- 应用ID
- 总案例数
- 待审核数
- 已通过数
- 已拒绝数
- 操作（查看案例）

**使用场景**：
- 了解各应用使用情况

### ⚙️ 系统设置

**显示内容**：

**信息包括**：

## 操作指南

### 🎯 日常审核流程

**标准流程**：
```
1. 登录后台
2. 点击"待审核"菜单
3. 查看待审核数量
4. 逐个点击"查看"预览内容
5. 合格的点击"通过"
6. 不合格的点击"拒绝"并填写原因
7. 统计数据自动更新
```

**批量审核流程**：
```
1. 在"待审核"页面
2. 勾选多个案例（复选框）
3. 或点击表头复选框全选
4. 点击"批量通过"或"批量拒绝"
5. 确认操作
6. 等待处理完成
7. 查看处理结果
```

### 🔍 搜索和筛选

**关键词搜索**：
```
1. 在搜索框输入关键词
2. 支持搜索：案例标题、应用ID
3. 点击"搜索"按钮
4. 查看搜索结果
```

**条件筛选**：
```
1. 选择"状态"下拉框
2. 选择：全部/待审核/已通过/已拒绝
3. 选择"应用"下拉框
4. 选择特定应用分类
5. 点击"搜索"应用筛选
```

**重置筛选**：
```
1. 点击"重置"按钮
2. 清空所有筛选条件
3. 显示全部案例
```

### 📊 分类管理

**查看应用统计**：
```
1. 点击"分类管理"菜单
2. 查看应用列表统计
3. 每个应用显示4个数量
   - 总案例数
   - 待审核数
   - 已通过数
   - 已拒绝数
```

**查看特定应用的案例**：
```
1. 在"分类管理"页面
2. 找到目标应用
3. 点击操作列的"查看案例"
4. 自动跳转到案例列表
5. 已应用筛选条件
```

### ⚙️ 系统管理

**修改密码**：
```
方式1：点击顶部"修改密码"按钮
方式2：在"系统设置"页面点击
方式3：点击侧边栏"退出登录"
```

**查看系统信息**：
```
1. 点击"系统设置"菜单
2. 查看站点、版本、路径信息
3. 查看Python和Flask版本
4. 检查系统运行状态
```

**快速操作**：
```
- 访问前台首页：在新标签页打开
- 修改管理员密码：打开修改密码弹窗
- 刷新系统：重新加载页面
```

## 界面元素说明

### 📊 统计卡片

```
┌────────────┐
│ 📊    25   │ ← 图标 + 数字
│ 总案例数   │ ← 标签
└────────────┘
```

**颜色含义**：
- 🔵 蓝色：总数（中性）
- 🟠 橙色：待审核（警告）
- 🟢 绿色：已通过（成功）
- 🔴 红色：已拒绝（错误）

### 📋 数据表格

**列说明**：
```
| 选择框 | 预览图 | 标题 | 应用ID | 状态 | 时间 | 操作 |
```

**状态标签**：
```
[待审核] - 黄色背景
[已通过] - 绿色背景
[已拒绝] - 红色背景
```

**操作按钮**：
- 查看（蓝色链接）- 打开详情页
- 通过（绿色链接）- 仅待审核显示
- 拒绝（红色链接）- 仅待审核显示
- 删除（红色链接）- 所有案例显示

### 🔍 搜索栏

```
[搜索输入框] [状态筛选▼] [应用筛选▼] [搜索] [重置]
```

**筛选选项**：
```
状态筛选：
  - 全部状态
  - 待审核
  - 已通过
  - 已拒绝

应用筛选：
  - 全部应用
  - 应用A（数量）
  - 应用B（数量）
  ...
```

### 📄 分页组件

```
共 100 条，第 1-20 条
< 上一页 [1] [2] [3] [4] [5] 下一页 >
```

**功能**：
- 显示总数和当前范围
- 上一页/下一页按钮
- 页码快速跳转
- 自动禁用边界按钮

## ⚡ 高效操作技巧

### 技巧1：快速审核
```
1. 使用"待审核"页面
2. 按"修改时间"排序
3. 从最早期的开始审核
4. 优先处理案例多的应用
```

### 技巧2：批量处理
```
1. 筛选到目标案例
2. 全选当前页
3. 批量通过/拒绝
4. 继续下一页
5. 重复步骤2-4
```

### 技巧3：分类管理
```
1. 进入"分类管理"
2. 找到待审核最多的应用
3. 点击"查看案例"
4. 集中审核该应用
5. 提高审核效率
```

### 技巧4：关键词搜索
```
1. 输入案例标题关键词
2. 或输入应用ID
3. 快速定位目标案例
4. 查看详情和状态
```

## 🚀 开始使用

### 1. 访问后台
```
URL: http://localhost:5000/admin
或: http://localhost:5000/admin/dashboard
```

### 2. 登录
```
如果未登录，会自动跳转到登录页面
输入管理员账号和密码
点击"登录"按钮
```

### 3. 查看数据
```
默认显示"工作台"
可看到4个统计卡片
下方显示所有案例列表
```

### 4. 切换页面
```
点击左侧菜单的不同选项
页面内容即时切换
统计数据实时更新
```

### 5. 进行操作
```
查看：点击"查看"按钮
审核：点击"通过"或"拒绝"
删除：点击"删除"按钮
批量：勾选多个案例后批量操作
```
